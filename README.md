# Markdown-Note

## 1、基本编辑

1. **粗体和斜体**
   
   源代码：
    ```
        *斜体*或_斜体_
        **粗体**
        ***加粗斜体***
        ~~删除线~~
    ```
   

    显示效果：

     >*斜体*或_斜体_\
        **粗体**\
        ***加粗斜体***\
        ~~删除线~~


2. **分级标题**
   * 第一种写法：
        ```
            这是一级标题
            ================
            这是二级标题
            ----------------
        ```

   * 第二种写法：
        ```
        # 一级标题
        ## 二级标题
        ### 三级标题
        #### 四级标题
        ##### 五级标题
        ###### 六级标题
        ```
3. **超链接**
   
   Markdown 支持两种形式的链接语法： 行内式和参考式两种形式，行内式一般使用较多。
    * 行内式：
        源代码：
        ```
            [链接文字](链接地址 "title属性")    注：title属性可加可不加

            不懂那就[百度一下](http://www.baidu.com)

            不懂那就[百度一下](http://www.baidu.com "梵居闹市")
        ```
        显示效果：
        
        >不懂那就[百度一下](http://www.baidu.com)\
            不懂那就[百度一下](http://www.baidu.com "梵居闹市")
