## 1、基本编辑
---------------------
1.　**粗体和斜体**

源代码：
```
    *斜体*或_斜体_
    **粗体**
    ***加粗斜体***
    ~~删除线~~
```
显示效果：
>*斜体*或_斜体_\
**粗体**\
***加粗斜体***\
~~删除线~~\

2. **分级标题**
   
* 第一种写法：
```
    这是一级标题
    ================
    这是二级标题
    ----------------
```
显示效果：

这是一级标题
================
这是二级标题
----------------

* 第二种写法：
  ```
    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
  ```
  显示效果：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
